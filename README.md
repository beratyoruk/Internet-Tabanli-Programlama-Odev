# Lezzet Durağı — Restoran Tanıtım Sitesi

İnternet Tabanlı Programlama dersi ödevi kapsamında hazırlanmış statik bir restoran tanıtım sitesi.

## Sayfalar

- **index.html** — Ana sayfa
- **hakkimizda.html** — Restoranın hikayesi
- **menu.html** — Yemek listesi ve fiyatlar
- **iletisim.html** — İletişim formu ve adres bilgileri

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Flexbox)
- Google Fonts (Nunito)

## Özellikler

- Flexbox ile oluşturulmuş 4 farklı sayfa düzeni
- Dışarıdan dahil edilen font (Google Fonts)
- İletişim formu (ad, e-posta, konu, mesaj)
- Tüm sayfalarda gezinme menüsü
